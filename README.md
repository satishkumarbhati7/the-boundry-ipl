## 🏟️ Features

### Hand-Crafted Cricket Stadium SVG Background
The hero section features a fully animated SVG cricket ground with:
- 🔦 Floodlight towers with glowing bulb arrays and light beams
- 🌿 Oval green field with mowing stripes
- 🏏 Detailed pitch with stumps & crease lines
- 🔴 Live animated cricket ball in play
- 📊 Scoreboard structure
- 👥 Stadium stands with coloured crowd dots
- ⭐ Twinkling stars in the night sky

All rendered in **pure SVG** — no external images needed!

### Admin Tabs

| Tab | What You Can Change |
|-----|---------------------|
| 🔴 **Today's Match** | Team names, live score, overs, venue, ticker message — updates hero live |
| 📅 **Upcoming Matches** | Add/remove match cards with dates, times & special offers |
| 🖼️ **Hero Banner** | Upload image file OR paste URL to overlay on stadium; edit all hero text |
| 📢 **Announcements** | Publish daily messages to the red top bar or offers page |
| 🎁 **Offers Text** | Edit and add match-based conditional discount offers |

> ✅ All changes reflect on the website **instantly** without a page reload.

## 🎮 Game Zone Logic

- Predict **Toss Winner** → +10 Points
- Predict **Match Winner** → +25 Points
- Predict **Highest Scorer** → +50 Points
- Redeem points for **discount coupons (10% – 50% OFF)**
- Live **leaderboard** with top predictors
